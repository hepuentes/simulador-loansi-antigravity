¿Cómo funciona?
  • Supervisores: Asignarles asesores. El supervisor verá las evaluaciones y simulaciones de esos asesores.
  • Gerentes: Asignarles supervisores. El gerente verá todo lo de los supervisores asignados Y sus asesores (jerarquía completa).
  • Auditores: Similar a gerentes, pueden tener supervisores asignados para auditar.
  • Si un usuario con permiso "equipo" no tiene asignaciones, verá 0 resultados.
  • Usuarios con permisos "todos" ven todo el sistema independiente de asignaciones.
Ejemplo de Jerarquía
Configuración Correcta:
  • Gerente1 → Supervisor1, Supervisor2
  • Supervisor1 → Asesor1, Asesor2, Asesor3
  • Supervisor2 → Asesor4, Asesor5

Gerente1 verá datos de los 5 asesores

Evitar:
  • No asignar un usuario a sí mismo
  • No crear ciclos (A→B→A)
  • No asignar gerentes a supervisores
